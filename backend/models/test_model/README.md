# Test Model Directory

This directory should contain the following files for the scGPT model:

1. `best_model.pt` - The trained model weights
2. `args.json` - Model configuration arguments
3. `vocab.json` - Gene vocabulary
4. `dev_train_args.yml` - Training arguments
5. `id2type.json` - Cell type ID to name mapping (optional)

Please add your actual model files to this directory before running the annotation pipeline.
